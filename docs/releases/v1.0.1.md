# Codex SyncBar 1.0.1

자동 인증 유지와 일회성 알림 동작을 더 안전하게 다듬은 패치 릴리즈입니다.

## 변경 사항

- 자동 인증 유지 중 실행 중인 ChatGPT와 Codex 데스크톱을 종료하지 않고, 활성 클라이언트가 끝날 때까지 갱신을 안전하게 보류합니다.
- 번들 ID를 기준으로 실행 중인 데스크톱 클라이언트를 구분하고 보류 상태를 화면에 표시합니다.
- `계정 별칭을 저장했습니다.` 같은 일회성 성공 알림은 한 번 표시된 뒤 팝오버나 설정창이 닫히거나 포커스를 잃으면 제거됩니다.
- 오류와 경고 알림은 포커스가 바뀌어도 유지되어 필요한 조치를 놓치지 않습니다.
- 실제 앱 화면을 바탕으로 개인정보가 없는 README UI 미리보기를 추가했습니다.

## 설치

1. `Codex-SyncBar-1.0.1-macOS-universal.zip`을 내려받아 압축을 풉니다.
2. `Codex SyncBar.app`을 `/Applications`로 옮깁니다.
3. 처음 한 번은 Finder에서 앱을 Control-클릭하고 **열기**를 선택합니다.

macOS 13 이상, `/Applications`의 Google Chrome과 공식 Codex CLI가 필요합니다. 이 ZIP은 Apple Silicon과 Intel Mac을 모두 포함한 유니버설 바이너리이며 ad-hoc 서명되어 있습니다.
