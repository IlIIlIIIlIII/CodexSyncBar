# Codex SyncBar 2.3.3

첫 공개 릴리즈입니다. 여러 ChatGPT/Codex 계정을 계정별 Chrome 세션으로 관리하고, 이 Mac과 등록한 SSH 장치에 한 번에 적용할 수 있습니다.

## 주요 기능

- 계정을 제한 없이 추가하고 5자 이하 별칭과 표시 순서를 관리할 수 있습니다.
- 5시간, 주간, Spark 5시간, Spark 주간 사용량을 선택해 표시합니다.
- 메뉴 막대에는 원하는 사용량 두 항목까지 간결하게 표시할 수 있습니다.
- 초기화권 수량과 다음 만료 시간을 별도 카드로 확인할 수 있습니다.
- SSH 장치별 연결 상태와 현재 적용 계정을 검증합니다.
- 최근 30일의 장치별 토큰 사용량과 공개 API 가격 기준 추정 비용을 집계합니다.
- 계정별 영구 Chrome 프로필로 Google 로그인, 패스키와 Touch ID를 지원합니다.
- 계정 전환 후 이전 인증을 캐시하는 원격 Codex app-server를 선별해 다시 시작합니다.

## 안전한 계정 전환

- 모든 장치를 먼저 점검한 뒤 변경하고, 실패하면 이전 상태로 복구합니다.
- Mac에만 전체 refresh token을 보관하고 SSH 장치에는 access-only 인증을 전달합니다.
- SSH 비밀번호와 키 암호는 macOS Keychain에만 저장합니다.
- 새 로그인 검증이 끝나기 전에는 기존 `auth.json`을 덮어쓰지 않습니다.
- 인증 파일의 소유권, 권한과 심볼릭 링크 여부를 확인하고 원자적으로 교체합니다.

## 설치

1. `Codex-SyncBar-2.3.3-macOS-universal.zip`을 내려받아 압축을 풉니다.
2. `Codex SyncBar.app`을 `/Applications`로 옮깁니다.
3. 처음 한 번은 Finder에서 앱을 Control-클릭하고 **열기**를 선택합니다.

macOS 13 이상, `/Applications`의 Google Chrome과 공식 Codex CLI가 필요합니다. 이 ZIP은 Apple Silicon과 Intel Mac을 모두 포함한 유니버설 바이너리이며 ad-hoc 서명되어 있습니다.

더 자세한 설정과 문제 해결 방법은 [README](https://github.com/IlIIlIIIlIII/CodexSyncBar#readme)를 참고하세요.
